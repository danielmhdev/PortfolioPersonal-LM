Mis Proyectos

Gestor de Tareas Personal

Aplicación de escritorio que permite crear, editar y organizar tareas diarias, asignar prioridades y marcar tareas completadas. Incluye búsqueda por fechas y categorías. Proyecto realizado en Java con interfaz gráfica usando JavaFX y almacenamiento local con SQLite.

Agenda de Contactos

App móvil desarrollada con Flutter que permite almacenar contactos personales o profesionales, con campos para nombre, teléfono, email, foto y notas. Incluye búsqueda rápida, filtros y edición de la agenda. Los datos se guardan localmente utilizando una base de datos SQLite.

Control de Inventario para Pequeños Negocios

Aplicación de escritorio para gestión de inventarios, registro de productos, entradas y salidas de stock y generación de reportes sencillos en PDF. Implementada en C# y .NET, con autenticación básica de usuarios y acceso a base de datos MySQL.

Lector de Noticias RSS

Programa multiplataforma (Java o Python) que permite agregar fuentes RSS, mostrar noticias actualizadas en una interfaz simple y marcar artículos como favoritos. Incluye almacenamiento de preferencias y estado de lectura.
